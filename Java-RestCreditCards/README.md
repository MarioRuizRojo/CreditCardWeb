# Credit card REST API
This credit card REST API server exposes all the necessary operations to manage a credit cards list.
Its written in java with Spring Boot framework, using a list type in-memory data structure.
Sofware tests performed with JUnit and Mockito library.

## getCreditCards endpoint
Operation to list all credit cards from the list
## addCreditCard endpoint
Adding a credit card to the list operation
## deleteCreditCard endpoint
Deleting a credit card from the list operation
## updateCreditCard endpoint
Updating a credit card from the list operation
